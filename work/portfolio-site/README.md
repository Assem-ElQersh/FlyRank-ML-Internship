# Portfolio Site

This directory contains the final portfolio for the FlyRank Internship "Ship the Ugly One" assignment. 

## Live Deployment

**Live URL:** [https://assemelqersh.netlify.app/](https://assemelqersh.netlify.app/)

*Note: I opted to deploy this project via Netlify rather than GitHub Pages. My custom domain (`assemelqersh.com`) is already pointed at a specific host for my original portfolio. Using GitHub Pages for this repository would have caused DNS routing conflicts with my existing live site. Netlify provided a clean, isolated deployment URL without disrupting my main domain's DNS records.*

> **Troubleshooting:** If you cannot open the Netlify link, please try using a VPN, as Netlify domains are blocked by ISPs in some countries.
